# Bellybutton Biodiversity Dashboard

## Overview

The Bellybutton Biodiversity Dashboard is a web application for exploring and visualizing belly button biodiversity data. It allows users to select a test subject, view demographic information, and explore interactive charts based on the dataset.

Selecting a Test Subject

Charts
Bar Chart:

Displays the top 10 Operational Taxonomic Units (OTUs) found in the selected individual.
X-axis: Sample Values
Y-axis: OTU IDs
Hovertext: OTU Labels
Bubble Chart:

Displays each sample.
X-axis: OTU IDs
Y-axis: Sample Values
Marker Size: Sample Values
Marker Colors: OTU IDs
Text Values: OTU Labels
Demographic Info:

Displays individual's demographic information below the charts.
